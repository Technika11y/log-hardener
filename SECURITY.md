# Security Policy

This tool reduces PII exposure in logs; it is a mitigation, **not** a guarantee of full
de-identification. Do not rely on it alone for regulated data — validate coverage against your own
data. A pattern that *misses* PII (under-redaction) is the high-severity failure mode here.

Report issues **in this tool** privately: GitHub **Report a vulnerability** on this repo, or email
the contact on the org profile. Target: **5 business days** to acknowledge, coordinated disclosure.
